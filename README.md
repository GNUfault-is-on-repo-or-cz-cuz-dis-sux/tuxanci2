# Tuxánci 2
Tuxánci 2 is modern successor to the orignal Tuxánci from 2007 made by Dušan Ďurech.

Tuxánci was a Czechoslovak multiplatform top-down deathmatch action game featuring Tux fighting another penguin with weapons.

Tuxánci was originally meant to be a Free Software clone of Bulánci, a Czech freeware action video game created in 2001 by SleepTeam. But then Tuxánci went its own way.
